Structure similarity search using the program DALI [ 18 ] showed that the S. typhimurium TSR structure closely resembles two other structures in Protein Data Bank ( PDB ) : tartronic semialdehyde reductase from S. typhimurium ( PDB id : 1YB4 ) and hydroxyisobutyrate dehydrogenase from Thermus thermophilus HB8 ( PDB id : 1J3 V ) .
